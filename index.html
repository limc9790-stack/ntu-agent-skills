03 — ANATOMY

Inside a SKILL.md

Every skill is just a folder with a SKILL.md file. Click each part to learn what it does.

YAML

Frontmatter — the "trigger"

▼
This is how Claude decides when to use your skill. The description field is the most important — Claude reads it and auto-loads the skill when your task matches.
--- name: my-skill description: Use when user asks about deployment or cloud ---
MARKDOWN

Instructions — what Claude does

▼
The body of SKILL.md is plain Markdown. Write step-by-step instructions, rules, examples, or anything you want Claude to follow every time this skill runs.
# Deploy Skill 1. Always run tests first 2. Use docker build 3. Push to registry 4. Run smoke test
SCRIPTS/

Python/Bash scripts

▼
Add executable scripts that Claude runs automatically. Only the output enters Claude's context — not the code itself. This saves tokens and keeps logic clean.
scripts/ validate.py # ← Claude runs this deploy.sh # ← and this
REFERENCES/

Extra documentation

▼
Put large docs, API references, or guidelines here. Claude loads them on demand — only when needed — so they don't waste context window space on every call.
references/ api-guide.md brand-colors.md db-schema.sql
ASSETS/

Templates & files

▼
Store output templates, fonts, images, or boilerplate files here. Claude references them by path — it copies and fills them in rather than generating from scratch.
assets/ report-template.docx logo.png email-template.html
WHERE

Where to put your skill

▼
Personal skills (all projects): ~/.claude/skills/
Project-specific: .claude/skills/ in your repo
Team/shared: commit .claude/skills/ to git
~/.claude/skills/ my-skill/ SKILL.md ← required scripts/ ← optional assets/ ← optional # 🆓 PATH A: Upload ZIP to claude.ai → Customize → Skills # 💳 PATH B: Claude Code (requires paid Claude Pro subscription) # 💳 PATH C: Claude Code + Anthropic direct ($20/mo Pro)

06 — HANDS ON

Try It Yourself

The free path is the browser upload (Path A). The terminal path (Path B) requires a paid Claude subscription.

🎉
Free path: Upload skill ZIP to claude.ai — works on ALL plans including free
Go to Settings → Capabilities → enable Code Execution, then Customize → Skills → Upload ZIP. Full Agent Skills work this way — no terminal, no subscription needed.
⚠️ Claude Code (the terminal tool) requires a paid Claude Pro subscription ($20/mo). The OpenRouter bypass no longer works in recent versions.
🆓 Path A — Upload to claude.ai (Free for ALL plans) NO TERMINAL · NO SUBSCRIPTION

Create your skill folder, zip it, upload to claude.ai → Customize → Skills. Claude uses it automatically in every chat. Works on phone, tablet, any browser.

1
Enable Code Execution in Settings
Skills require code execution to run. Enable it once and it stays on.
claude.ai → Settings (top right) → Capabilities → toggle ON "Code execution and file creation" ✓
2
Create your SKILL.md file
Use the Skill Builder in Section 04 above, copy the output, save as a file named SKILL.md. Or use the EDA Analyst skill from Section 07.
File: SKILL.md --- name: eda-analyst description: Use when user asks to analyse a dataset, do EDA, explore data, check data quality, or prepare data for machine learning. --- # EDA Analyst When I share a dataset, always: 1. Identify missing values + suggest imputation 2. Flag class imbalance (any class under 15%) 3. Report outliers (Z-score > 3) 4. Show top 3 correlations with target column 5. Give 3 actionable recommendations 6. Format as a structured report Tone: technical but friendly — explain WHY.
3
Create a folder and ZIP it
Put your SKILL.md inside a folder. The folder name = skill name. Then zip the folder.
📁 eda-analyst/ ← folder name = skill name └── SKILL.md ← your instructions Right-click folder → "Compress" / "Send to ZIP" → produces: eda-analyst.zip ✓
4
Upload to claude.ai Customize → Skills
Navigate to the Skills page, click +, upload your ZIP file.
claude.ai → Customize → Skills → click "+" → "Upload a skill" → select eda-analyst.zip → Toggle skill ON ✓
5
Test it — start a new chat and upload the dataset
Download student_grades.csv from Section 07C below, upload it in a new chat, and type:
Please analyse this dataset.
Claude detects the task matches your skill's description → activates automatically → runs your EDA instructions.
✅ What you get: Real Agent Skills — Claude auto-activates your EDA skill on any relevant task. Works in every chat, on any device, completely free.
💡 To add scripts/ or references/: Include them in the same folder before zipping. The full 4-folder structure from Section 07 works exactly the same when uploaded — no Claude Code needed.

07B — HOW THE 4 FILES WORK TOGETHER

When you say "analyse my dataset", here's the chain reaction — in order.

📄
1. SKILL.md triggers automatically
Claude reads your message and matches it to the description field. The skill activates — no slash command needed.
↓
🐍
2. scripts/analyze.py runs on your CSV
Python script executes on your dataset. Only the output (stats, missing values, correlations) enters the conversation — not the 40 lines of code. Saves tokens.
↓
📋
3. references/ds-checklist.md loaded
Claude reads your team's DS quality checklist — class imbalance thresholds, leakage rules, outlier guidelines. Loaded on-demand only. Zero token waste when not needed.
↓
📝
4. assets/report-template.md filled in
Claude fills the template with actual findings. Same professional structure every time — whether it's your dataset, a teammate's, or a Kaggle CSV.
↓
✅
Result: Complete EDA report in seconds
Structured, professional, consistent — and zero repeated instructions from you. The skill remembers everything.
